This is an web application THE QUIZZER, created by me. It is an university project for Web Development course.
Purpose of this application is users to test their skills in other people's quizzes. You can create your own quiz and share it to whoever you like. You can also collect points for completing, or creating, quizzes and get placed on the global leaderboard. Have fun.
